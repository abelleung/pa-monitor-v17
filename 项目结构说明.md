## 各文件功能详解

| 文件 | 功能概要 |
|------|----------|
| pa_monitor.py | **核心监控程序**。中国平安(601318) T+0 三策略实盘监控：倒T（动量/BOLL/冲高回落）+ 正T。含信号触发、止盈止损、90分钟窗口评估、心跳推送、仓位追踪。从pytdx拉取1分钟K线，盘中持续运行。 |
| pa_api_server.py | **API服务器**（运行在腾讯云）。Flask + CORS，为 abel-leung.com 提供K线数据、信号历史、交易统计等REST接口。 |
| pa_position_tracker.py | **持仓跟踪系统**。记录每次截图持仓数据，对比成本变化，推算不同股数下的等效成本，计算做T效果和成本递减进度。支持双口径盈亏统计（实际vs理论信号有效性）。 |
| pa_notify.py | **推送通知模块**。支持 Bark(iOS)、ntfy(Android)、企业微信机器人三种渠道。强制不走代理，确保推送可靠。 |
| daily_report_enhanced.py | **增强日报生成**。收盘后自动生成四图（价格/BOLL/MACD/成交额）、三策略条件满足度分析、明日交易建议，推送带图报告。 |
| run_daily_report.py | **收盘复盘入口**。15:05定时运行，从pytdx拉取当日K线 → 调用日报生成 → 推送报告到Bark+ntfy。 |
| realtime_data_feeder.py | **独立实时数据服务**。每15秒从腾讯分时接口拉取数据，写入 realtime_bars.json，与 pa_monitor 解耦可独立重启。 |
| fetch_and_sync_1min.py | **历史数据同步**。从云端拉取 pa_monitor 导出的1分钟CSV，或补全本地缺失数据（从桌面历史CSV拆分）。 |
| rss_filter.py | **RSS筛选器**。从Miniflux拉取文章，按关键词（AI工具/投资量化/管理认知/心理学/科技商业）分类筛选，保存Markdown，不推送避免干扰盯盘。 |
| index.html | **前端页面**。暗色/亮色主题，展示实时K线、交易信号、统计数据，使用 JetBrains Mono + Noto Sans SC 字体。 |
| setup_monitor.sh | **管理脚本**。install/uninstall/start/stop/status/test/logs 子命令，配置 launchd + cron 自检。 |
| monitor_healthcheck.sh | **健康检查**。在关键时间点（09:25/09:35/08:50/12:50）检查监控进程，心跳超时15分钟则重启。 |
| deploy.sh | **一键部署**。上传代码到云端服务器，更新systemd服务版本号，清除matplotlib缓存，重启服务。 |
| version_manager.sh | **版本管理**。backup/rollback/list/current 子命令，管理版本备份目录。 |

## 技术栈

- **数据源**：pytdx（通达信行情）、腾讯分时接口
- **策略**：三策略倒T（动量/BOLL上轨/冲高回落）+ 正T极低位高胜率组合
- **后端**：Python 3 + Flask + pandas + matplotlib
- **推送**：Bark（iOS）、ntfy（Android）、企业微信Webhook
- **前端**：纯HTML/CSS/JS，无框架依赖
- **部署**：本地（macOS launchd）+ 云端（腾讯云 systemd）
